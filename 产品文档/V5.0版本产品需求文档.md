## 3. UI/UE 设计要求

### 3.1 整体风格

- **现代科技感**：采用渐变色、玻璃拟态、微动画
- **深色主题**：默认深色模式，护眼
- **渐变色彩**：蓝紫渐变为主，活力十足

### 3.2 登录/注册页面

- 全屏背景动画（粒子效果或波浪）
- 玻璃拟态登录框
- 输入框带图标和动画反馈
- 密码强度指示器
- 记住密码/忘记密码选项

### 3.3 主页面

- 顶部导航栏：毛玻璃效果
- 题目卡片：悬停动画、展开动画
- Tab切换：滑动指示器
- 收藏/复习：点击反馈动画
- AI问答：气泡动画、输入动画

### 3.4 响应式设计

- 移动端适配（375px+）
- 平板适配（768px+）
- 桌面适配（1024px+）

## 4. 技术架构

### 4.1 前端

| 技术 | 用途 |
|------|------|
| React 18 | UI框架 |
| Framer Motion | 动画库 |
| Tailwind CSS | 样式 |
| React Router | 路由 |

### 4.2 后端

| 技术 | 用途 |
|------|------|
| Supabase | 认证+数据库 |
| Edge Functions | API |

### 4.3 数据库设计

```sql
-- 用户表
users (
  id: uuid,
  email: string,
  password_hash: string,
  nickname: string,
  created_at: timestamp,
  updated_at: timestamp
)

-- 收藏表
favorites (
  id: uuid,
  user_id: uuid,
  question_id: number,
  created_at: timestamp
)

-- 复习记录表
reviews (
  id: uuid,
  user_id: uuid,
  question_id: number,
  created_at: timestamp
)

-- 问答历史表
chat_history (
  id: uuid,
  user_id: uuid,
  messages: jsonb,
  created_at: timestamp
)

-- 用户配置表
user_config (
  id: uuid,
  user_id: uuid,
  resume: text,
  target_position: string,
  updated_at: timestamp
)
```
