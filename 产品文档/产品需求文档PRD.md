# AI 面试模拟器 - 产品需求文档 (PRD)

**产品名称**：AIInterviewer - AI产品面试神器  
**版本**：V1.0  
**状态**：规划中  
**负责人**：刘思成

---

## 1. 产品概述

### 1.1 产品定位
专注于 AI 产品/运营 求职者的面试准备工具，帮助用户快速掌握 AI 产品经理面试高频问题及答题方法论。

### 1.2 核心价值
- **对于求职者**：告别盲目刷题，提供 AI 方向的精准面试准备
- **对于面试官**：提供 AI 产品能力评估标准

### 1.3 目标用户
| 用户群体 | 特征 | 痛点 |
|----------|------|------|
| AI产品求职者 | 0-3年经验 | 找不到AI方向面试资料 |
| 转型AI的传统PM | 有PM经验 | 缺乏AI产品项目经验 |
| AI运营求职者 | 0-3年经验 | 不懂AI业务怎么面 |

---

## 2. 核心功能矩阵

### 2.1 功能列表

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 简历解析 | 解析用户简历，提取关键信息 | P0 |
| 岗位匹配 | 根据目标岗位生成定制化问题 | P0 |
| 题库浏览 | AI产品/运营高频面试题库 | P0 |
| 答案参考 | 提供答题思路和参考回答 | P0 |
| 收藏夹 | 收藏重点题目 | P1 |
| 面试记录 | 记录面试准备进度 | P2 |

### 2.2 题目类型覆盖

| 类别 | 示例 |
|------|------|
| 产品设计类 | "如何设计一个AI推荐系统？" |
| 数据分析类 | "模型效果下降30%怎么分析？" |
| 项目经验类 | "介绍一下你做过的AI项目" |
| AI基础类 | "什么是RAG？和微调的区别？" |
| 业务场景类 | "如何提升AI产品的用户留存？" |

---

## 3. 用户流程

```
用户进入 → 选择目标岗位 → 输入简历/经历 → 
生成定制问题 → 查看问题 + 参考答案 → 收藏/标记 → 
模拟回答 → 查看反馈
```

## 4. 迭代版本规划

### V1.0 MVP（1-2周）

**目标**：验证核心需求，跑通最小闭环

| 功能 | 说明 |
|------|------|
| 题库浏览 | 30-50道高频AI产品面试题 |
| 分类筛选 | 按类型（产品/数据/基础）筛选 |
| 参考答案 | 每题附带答题思路 |
| 静态页面 | Web页面，无用户系统 |

**交付物**：可访问的 Web 页面，直接可用

**验证指标**：UV、停留时长

---

### V2.0 交互版（2-3周）

**目标**：增加交互能力，提升用户参与度

| 功能 | 说明 |
|------|------|
| 简历解析 | 粘贴简历，提取关键经历 |
| 岗位匹配 | 输入目标岗位，生成定制问题 |
| 收藏功能 | 收藏重点题目（本地存储） |
| 进度记录 | 记录已复习题目 |

**技术升级**：引入简单的用户系统（LocalStorage）

**验证指标**：收藏率、生成次数

---

### V3.0 智能版（3-4周）

**目标**：AI 能力接入，实现智能化

| 功能 | 说明 |
|------|------|
| AI 生成答案 | 用 LLM 生成个性化参考答案 |
| 答题模拟 | 用户语音/文字回答，AI给出反馈 |
| 智能推荐 | 根据用户表现推荐薄弱环节 |

**技术升级**：接入 LLM API（GPT/Claude）

**验证指标**：付费转化、用户留存

---

### V4.0 社区版（4-6周）

**目标**：引入社交功能，提升粘性

| 功能 | 说明 |
|------|------|
| 面试搭子 | 匹配其他求职者互相模拟 |
| 面经分享 | 用户分享真实面试经验 |
| 评论讨论 | 对题目进行讨论 |
| 排行榜 | 复习进度排行 |

**技术升级**：后端服务、数据库、用户体系

**验证指标**：DAU、UGC数量

---

### V5.0 商业化版（6-8周）

**目标**：验证商业模型

| 功能 | 说明 |
|------|------|
| 付费题库 | 高级题目（99+道） |
| 1v1模拟 | AI面试官深度模拟 |
| 简历优化 | AI辅助简历修改 |
| 企业版 | 招聘方使用 |

**商业模式**：
- 免费版：基础题库（50道）
- 付费版：高级题库 + AI模拟（29.9元/月）
- 企业版：定制题库（999元/月）

---

### V6.0 规模化版（2-3个月）

**目标**：扩展品类，规模化增长

| 功能 | 说明 |
|------|------|
| 多岗位支持 | AI运营、算法工程师、数据分析师 |
| 多语言 | 英文面试准备 |
| 小程序 | 微信小程序上线 |
| API开放 | 提供面试API给第三方 |

## 5. 技术架构

### 5.1 技术选型

| 层级 | 技术 |
|------|------|
| 前端 | React + Vite / Next.js |

- **敏捷开发**：2周一个Sprint

- **快速迭代**：优先做高价值功能

## 6. 风险与对策

| 风险 | 对策 |
|------|------|
| LLM API 成本超支 | 限制免费用户调用次数，设置付费墙 |
| 内容同质化 | 持续产出独家面试题，保持差异化 |
| 用户留存低 | 增加社区功能，提升粘性 |
| 竞品复制 | 快速迭代，先发优势 + 社区壁垒 |
