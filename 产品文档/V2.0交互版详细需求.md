# AI面试模拟器 - V2.0 交互版 详细需求

**版本**：V2.0
**周期**：2-3周
**目标**：增加用户交互，收集用户数据

---

## 1. 功能范围

### 1.1 新增功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 简历解析 | 粘贴简历，提取关键经历，生成个性化问题 | P0 |
| 岗位匹配 | 输入目标岗位，智能推荐相关题目 | P0 |
| 收藏功能 | 点击收藏题目，本地存储 | P0 |
| 进度记录 | 记录已复习题目，本地存储 | P1 |

### 1.2 优化功能

| 功能 | 描述 |
|------|------|
| 题库扩展 | 从30道扩展到50道 |
| 智能推荐 | 基于用户收藏和岗位推荐题目 |

## 3. 用户交互流程

```
用户进入 
  ↓
【可选】填写简历 + 目标岗位 → 生成定制推荐
  ↓
浏览/搜索题目
  ↓
点击题目 → 展开答案
  ↓
收藏 / 标记已复习
  ↓
查看收藏列表 / 复习进度
```
