# V5.0 酷炫登录页面 - 设计稿

## 页面1：登录页面

### 设计要求

**整体风格**：
- 深色背景 + 渐变蓝色/紫色动画
- 玻璃拟态（Glassmorphism）卡片
- 霓虹光效点缀
- 流畅动画

**具体设计**：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│         🌊 动态渐变背景 (深蓝→紫→青)               │
│                                                     │
│    ┌─────────────────────────────────────────┐      │
│    │                                         │      │
│    │     🤖  AI产品面试题库                  │      │
│    │     ─────────────────────               │      │
│    │                                         │      │
│    │     📧  ┌─────────────────────┐        │      │
│    │         │ 输入邮箱地址        │        │      │
│    │         └─────────────────────┘        │      │
│    │                                         │      │
│    │     🔒  ┌─────────────────────┐        │      │
│    │         │ 输入密码            │        │      │
│    │         └─────────────────────┘        │      │
│    │                                         │      │
│    │     ☐ 记住密码          忘记密码？    │      │
│    │                                         │      │
│    │     ┌─────────────────────────────┐      │      │
│    │     │        登 录              │      │      │
│    │     └─────────────────────────────┘      │      │
│    │                                         │      │
│    │     还没有账号？ 立即注册 →            │      │
│    │                                         │      │
│    └─────────────────────────────────────────┘      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**动画效果**：
- 背景：缓慢流动的渐变色波浪
- 输入框：聚焦时边框发光（蓝→紫渐变）
- 按钮：悬停时发光、点击时缩放
- 页面加载：卡片从下方滑入

## 设计规范

### 颜色

| 用途 | 颜色 |
|------|------|
| 主色 | #6366F1 (Indigo-500) |
| 渐变起 | #8B5CF6 (Violet-500) |
| 渐变止 | #06B6D4 (Cyan-500) |
| 背景 | #0F172A (Slate-900) |
| 卡片背景 | rgba(255,255,255,0.1) |
| 文字 | #F8FAFC (Slate-50) |
| 次要文字 | #94A3B8 (Slate-400) |

### 字体

- 主字体：Inter, system-ui
- 标题：Bold 700
- 正文：Regular 400

### 动画时长

- 快速：150ms
- 中速：300ms
- 慢速：500ms
