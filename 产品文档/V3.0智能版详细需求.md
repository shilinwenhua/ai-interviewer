# AI面试模拟器 - V3.0 智能版 详细需求

**版本**：V3.0
**周期**：3-4周
**目标**：接入AI能力，实现智能化问答

---

## 1. 功能范围

### 1.1 核心功能（必须）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| AI问答 | 基于LLM生成个性化参考答案 | P0 |
| 追问功能 | 用户可以追问，AI深入解答 | P0 |
| 语音输入 | 支持语音转文字提问 | P1 |

### 1.2 新增功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 答题模式 | 用户先回答，AI再点评 | P1 |
| 智能推荐 | 根据用户薄弱环节推荐题目 | P2 |

## 3. AI问答流程

```
用户输入问题 
  ↓
调用LLM API（带上系统提示词）
  ↓
返回AI回答
  ↓
用户可以继续追问
  ↓
记录问答历史
```

## 5. 技术实现

### 5.1 LLM接入

| 方案 | 优点 | 缺点 |
|------|------|------|
| OpenAI API | 效果好，全球通用 | 需要翻墙 |
| Claude API | 效果好 | 需要翻墙 |
| 国产LLM | 国内可用 | 效果稍差 |

### 5.2 实现方式

- 前端调用后端API（或直接调用）
- 使用 Vercel Edge Functions / Cloudflare Workers 部署后端
- 关键：保护API Key
