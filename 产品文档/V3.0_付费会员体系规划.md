**版本**：V3.0

## 1. 商业模式

### 1.1 核心逻辑

```
免费用户 → 试用/导流 → 付费转化 → 会员留存 → 复购/裂变
```

### 1.2 定价策略

| 方案 | 价格 | 定位 |
|------|------|------|
| 免费版 | ¥0 | 获客 + 试用 |
| 月卡 | ¥29/月 | 低门槛尝试 |
| 年卡 | ¥199/年 | 主力变现（推荐） |
| 永久卡 | ¥499 | 铁杆粉丝 |

## 3. 转化策略

### 3.1 免费转付费路径

```
1. 用户注册 → 免费使用基础功能
2. 每天 1 次 AI 面试 → 体验 AI 反馈（模糊版）
3. 想看详细反馈 → 提示"开通会员查看完整反馈"
4. 引导至支付页面 → 选择月卡/年卡
```

### 3.2 会员权益包装

| 权益 | 免费版 | 会员版 |
|------|--------|--------|
| AI 面试次数 | 1次/天 | 无限 |
| AI 反馈详细度 | 粗糙 | 完整 5 维度 |
| 题库数量 | 30 | 500+ |
| 公司覆盖 | 3 家 | 20+ 家 |
| 简历定制 | ❌ | ✅ |
| 报告导出 | ❌ | ✅ |
| 语音答题 | ❌ | ✅ |
