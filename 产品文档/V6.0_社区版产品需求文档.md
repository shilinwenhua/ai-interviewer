# AI面试题库 V6.0 版本产品需求文档

**版本**：V6.0  
**代号**：社区版  
**日期**：2026-03-03

---

## 1. 版本定位

V6.0 是从"工具"到"社区"的关键版本，核心目标是：

1. **每日精选** - 用户问AI的优质问题，自动沉淀到题库
2. **社区广场** - 打造AI产品面试领域的垂直社区
3. **酷炫AI感UI** - 全新设计语言，极具科技感

---

## 2. 核心功能

### 2.1 每日精选（Daily Picks）

| 功能 | 描述 |
|------|------|
| AI问答日志 | 记录用户所有问答，沉淀优质内容 |
| 精选问题池 | 每日由运营/规则筛选3-5道优质问题 |
| 精选展示 | 首页"精选"Tab，展示最新精选 |
| 点赞收藏 | 用户可以对精选问题点赞收藏 |

### 2.2 社区广场（Community）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 问题列表 | 浏览所有社区问题 | P0 |
| 分类筛选 | 按类别/状态筛选问题 | P0 |
| 搜索 | 搜索问题和回答 | P0 |
| 提问 | 用户发起新问题 | P1 |
| 回答 | 用户回复解答 | P1 |
| 采纳 | 提问者采纳最佳回答 | P1 |
| 点赞 | 为优质回答点赞 | P1 |

### 2.3 用户系统

| 功能 | 描述 |
|------|------|
| 邮箱注册 | 邮箱+密码注册 |
| 邮箱登录 | 邮箱+密码登录 |
| 个人主页 | 查看我的提问/回答 |
| 我的积分 | 社区贡献积分 |

## 3. UI/UE 设计要求

### 3.1 设计理念

**"AI原生感"** - 让用户一看就知道是AI时代的产品

### 3.2 设计关键词

| 关键词 | 含义 |
|--------|------|
| 流动感 | 渐变、动画、像AI在思考 |
| 未来感 | 深色、霓虹、科技感 |
| 呼吸感 | 留白、节奏、像生命体 |

## 5. 数据模型

### 5.1 用户表（users）

```sql
id: uuid
email: string (unique)
password_hash: string
nickname: string
avatar_url: string
points: integer (default: 0)
created_at: timestamp
```

### 5.2 问题表（questions）

```sql
id: uuid
user_id: uuid (外键)
title: string
content: text
category: string
tags: text[]
status: enum (open/answered/resolved)
is_picked: boolean (是否被精选)
view_count: integer
answer_count: integer
created_at: timestamp
```

### 5.3 回答表（answers）

```sql
id: uuid
question_id: uuid (外键)
user_id: uuid (外键)
content: text
is_accepted: boolean (是否被采纳)
upvotes: integer
created_at: timestamp
```

### 5.4 精选日志表（picks）

```sql
id: uuid
question_id: uuid (外键)
pick_date: date
period: integer (第几期)
```

### 5.5 问答日志表（chat_logs）

```sql
id: uuid
user_id: uuid (外键)
user_message: text
ai_response: text
created_at: timestamp
```

## 9. 设计参考

### 9.1 颜色规范

| 用途 | 颜色 | 效果 |
|------|------|------|
| 主色 | #6366F1 (Indigo) | 科技感 |
| 渐变起 | #8B5CF6 (Violet) | 神秘感 |
| 渐变止 | #06B6D4 (Cyan) | 未来感 |
| 背景 | #0A0A0F | 深邃 |
| 卡片 | rgba(255,255,255,0.05) | 玻璃感 |
| 文字主 | #FFFFFF | 清晰 |
| 次要 | #94A3B8 | 柔和 |

### 9.2 动画时长

| 类型 | 时长 | 效果 |
|------|------|------|
| 快速 | 150ms | 点击反馈 |
| 中速 | 300ms | 过渡 |
| 慢速 | 500ms | 页面切换 |
| AI思考 | 1000ms | 打字效果 |

### 9.3 组件规范

- 圆角：12px-24px（柔和）
- 间距：4px-32px（呼吸感）
- 阴影：0 0 30px rgba(99,102,241,0.3)（发光）
