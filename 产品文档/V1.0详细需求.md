# AI面试模拟器 - V1.0 MVP 详细需求

**版本**：V1.0
**周期**：1-2周
**目标**：快速验证需求，产出可演示的静态页面

---

## 1. 功能范围

### 1.1 核心功能（必须）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 题库浏览 | 展示30-50道AI产品面试题 | P0 |
| 分类筛选 | 三个Tab：产品问题 / 数据问题 / 基础概念 | P0 |
| 查看答案 | 点击题目展开参考答案和答题思路 | P0 |

### 1.2 附加功能（尽量有）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 搜索 | 关键词搜索题目 | P1 |
| 暗黑模式 | 切换深色/浅色主题 | P1 |

---

## 2. 题目分类定义

### 2.1 产品设计类 (Product Design)
- AI产品设计思路
- 用户需求分析
- 功能优先级排序
- 竞品分析

### 2.2 数据分析类 (Data Analysis)
- 指标异常分析
- AB测试设计
- 效果评估方法
- 数据驱动决策

### 2.3 AI基础概念类 (AI Fundamentals)
- 机器学习基础
- 模型评估指标
- RAG/微调等概念
- 算法原理

---

## 3. 题目数据格式

```json
{
  "id": 1,
  "category": "product",
  "question": "问题内容",
  "answer": "参考答案/答题思路",
  "difficulty": "easy/medium/hard",
  "tags": ["标签1", "标签2"]
}
```

## 5. 技术实现

### 5.1 技术栈
- **框架**：React + Vite
- **样式**：Tailwind CSS
- **状态**：React Hooks
- **部署**：Vercel / Cloudflare Pages
