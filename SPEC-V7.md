# AI面试题库 V7.0 - 迭代规格

## 目标
增加面试记录 + 个性化题库推荐，提升用户粘性

---

## 功能1：面试记录系统

### 1.1 数据结构
```javascript
// 面试记录
interviewHistory: [
  {
    id: 'xxx',
    date: '2026-03-05',
    role: 'pm',
    roleName: '产品经理',
    questions: [
      {
        question: '问题内容',
        answer: '用户回答',
        feedback: { score: 85, strengths: '...', improvements: '...' }
      }
    ],
    averageScore: 82,
    duration: 15, // 分钟
  }
]
```

### 1.2 个人中心入口
- 显示"面试记录" tab
- 列表展示历史面试（日期、岗位、评分）
- 点击查看详情

### 1.3 能力雷达图
- 维度：自我介绍、需求分析、框架原理、编码能力、协作能力等
- 根据历史面试数据生成雷达图
- 对比上次 vs 本次

---

## 功能2：个性化题库

### 2.1 岗位进阶选择
- 基础岗位：产品经理、前端、后端、运营
- 进阶选项：选择目标公司类型（大厂/中小厂/创业）
- 经验级别：校招/1-3年/3-5年/5年+

### 2.2 题库分类
- 通用题：所有岗位通用
- 专有题：按岗位细分
- 大厂真题：按公司分类（字节、阿里、腾讯等）
- 高频题：用户贡献的常见题

### 2.3 AI智能推荐
- 分析用户历史面试的薄弱环节
- 推荐相关强化题目
- "为我推荐"按钮

---

## 技术实现

### 文件结构
- components/InterviewHistory.jsx - 面试记录页面
- components/AbilityRadar.jsx - 能力雷达图
- components/QuestionBank.jsx - 个性化题库
- components/RecommendedQuestions.jsx - AI推荐题目

### 数据存储
- 使用 localStorage 存储用户数据
- 后续可迁移到后端

---

## 验收标准
1. 用户可以开始AI面试，面试结束后记录被保存
2. 用户可以在个人中心查看历史面试记录
3. 能力雷达图正确显示各维度得分
4. 题库按岗位分类，用户可选择不同题库
5. "为我推荐"功能能根据历史薄弱环节推荐题目
